# Stage_M2
Subject : diversity and the evolutionary history of bacterial symbiotes Serratia symbiotica and Buchnera aphidicola in the pea aphid (Acyrthosiphon pisum).

This GitHub lists a set of key scripts used to perform my analyses.

**Mapping** : Bash script used to map the genomes of *Serratia* and *Buchnera* (see mapping/README.md)

**Phylogeny** : Bash script that enabled the reconstruction of symbiotic phylogenies (see phylogeny/README.md)

**Unused** : Set of R scripts containing analyses that were not used or did not work (see unused/README.md)

**Script_traitement_symbiote** : R Script grouping all the R analyses used during the internship and present in the report. The script includes a part with all the functions used, a part to load the data (to be adapted as needed), then a part for each analysis carried out: statistical analysis (PGLS, GLM), visualization of phylogenetic trees, visualization of data explaining the presence of the symbiote *Serratia* (biotype, geographical area, climate), Reconstruction of ancestral characters, table comparing the evolution rates of symbionts, measure of congruence with Parafit, and co-phylogeny

**Verif_qualite_seq** : Script R to analyze the quality of DNA sequence based on the undetermined base percentage. Produce tables with this percentage for each individual.
