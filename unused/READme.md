All the scripts not presented in the report 

PACo. R: measures the congruence between two phylogenetic trees 

Annotation. R: Process the annotations made on Genious, list the genes found and whether they are present in all individuals.

Clean_name_leaf. A: Clean the names of tree leaves to make them homomogenic.

Script_16S.R: Processing of 16S data, cleaning by removing the blaster symbiote less than x times, which symbiote is found in which individual according to 16S. 

Scipt_ancr. R: Reconstruction of ancestral characters by phytools.

Regression : produces a linear regression of substitutions by site for *Buchnera* and according to that of *Serratia*. Site substitutions equal to the branch length leading to the individual’s leaf.
